# Vimrc

## Usage

### Install Vim

```console
brew install vim --with-lua
```

### Clone this repo

```console
git clone https://github.com/syumai/vimrc.git ~/.vim
```

### Install plugins

#### Install vim-plug

```console
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

#### Install plugins

```console
vim

:PlugInstall
```
